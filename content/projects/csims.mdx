# CSIMS – Institutional Website

Modern institutional website focused on clarity, performance, and accessibility. I contributed to
page structure, componentization, and responsive layouts using Next.js and Tailwind CSS.

## What I Built

- **Reusable Components**: Built a comprehensive component library for consistent UI across all pages
- **Responsive Layouts**: Ensured seamless experience across desktop, tablet, and mobile devices
- **Information Architecture**: Improved navigation and page structure for content-heavy sections
- **SEO & Accessibility**: Applied semantic HTML, proper headings, and alt text for better discoverability

## Impact

- Clean, maintainable component system that scales with content growth
- Improved user experience through better navigation and responsive design
- Enhanced accessibility compliance for broader audience reach
- SEO-optimized structure for better search engine visibility

## Technologies Used

- **Frontend**: Next.js 14, TypeScript, React
- **Styling**: Tailwind CSS for consistent design system
- **Performance**: Optimized images and code splitting
- **Accessibility**: Semantic HTML and ARIA labels
