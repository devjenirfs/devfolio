# Contreras Law – Law Firm Website

A fast and reliable website for a law firm. I designed the information architecture,
implemented the UI with Next.js and Tailwind CSS, and set up SEO foundations for better
discoverability and trust.

## What I Built

- **Information Architecture**: Designed intuitive site structure and navigation for legal services
- **UI Design**: Created clean, professional interface that builds trust and credibility
- **Responsive Design**: Ensured seamless experience across all devices and screen sizes
- **SEO Foundations**: Implemented meta tags, structured data, and sitemap for better search visibility

## Impact

- **Core Web Vitals**: Lighthouse-friendly structure with optimal performance scores
- **Lead Generation**: Improved contact forms and call-to-action placement
- **Professional Presence**: Clean, accessible design that reflects firm's expertise
- **Maintainability**: Solo-built system that's easy to update and maintain

## Technologies Used

- **Frontend**: Next.js 14, TypeScript, React
- **Styling**: Tailwind CSS for rapid, consistent development
- **Performance**: Optimized images, lazy loading, and code splitting
- **SEO**: Next.js built-in SEO features and custom meta management
